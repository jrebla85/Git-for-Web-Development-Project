1. What is Semantic HTML? 

Semantic html is html used to convey meaning like <header> or <footer>. It has no impact on the page itself, its mostly used to let yoruself and others know what section of the page they are working on.
    
2. What is HTML used for? 

HTML is like the framing of a house, it organizes the information and makes it readable.
    
3. What is an attribute and where do we put it? 

An html attribute provides additional information about a tag. It is put in the opening tag.

4. What is the h1 tag used for? How many times should I use it on a page?

The h1 tag is typically the title of the page, and therefore should only be used once.
    
5. Name two tags that have required attributes

<a> and <img> which require href and src respectively.
    
6. What do we put in the head of our HTML document? 

Metadata such as the page title, location of the style sheets and etc should be located in the head.

7. What is an id? 

An id is a unique identifier that can be added to an html element. It is more specific than a class and takes priority.
    
8. What elements can I add an id to? 

Any element, however an id can only be used once.
    
9. How many times can I use the same id on a page? 

Id's may not be reused.
    
10. What is a class? 

A class is like an ID, but is less specific and can be used in multiple html elements.
    
11. What elements can I add a class to? 

Any element, or multiple elements and multiple times.
    
12. How many times can I use the same class on a page? 

As many as needed.
    
13. How do I get my link to open in a new tab?

by adding target="blank" inside the opening anchor tag.
    
14. What is the alt attribute in the image tag used for? 

Alt attribute is used to display a specified message int he event that an image cannot be displayed. It is also used by screen readers so is useful for those who may nto be able to see the image.
    
15. How do I reference an id?

19. What is the difference between a class and an id? - Give me an example of when I might use each one

A class is less specific than an ID. For instance a class could be used multiple times for section headers <h2 class="whatever> multiple times. However, with an ID it can be used only once for intance with a certain paragraph you wanted to style differently than the rest <p id="whatever>.
